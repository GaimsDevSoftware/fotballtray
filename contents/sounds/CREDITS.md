# Sound credits

The current sound set was carried over from the earlier "sportsbar" applet and
trimmed/normalised (mono 44.1 kHz, loudnorm I=-16) for this widget.

| File | What | Origin |
|------|------|--------|
| `cheer.wav` | Crowd cheer (your team's goal) | crowd-cheering recording, trimmed to 3.6 s + fade |
| `whistle.wav` | Short referee whistle (goal alert) | referee-whistle recording |
| `whistle_match.wav` | Long referee whistle (kickoff / full-time) | game-over whistle recording |
| `bong.wav` | The classic "Tippekampen" bong | tippekampen-dooong recording |
| `pling.wav` | Soft notification ping | tippekamp-pling recording |

Provenance of the original recordings is not fully documented. Before publishing
the widget, replace any non-CC0 clips with verified CC0 assets (freesound.org
filtered to CC0) to keep the whole package redistributable. Referee whistles can
also be synthesised (a pea-whistle is ~3.5 kHz fundamental + harmonics + ~18 Hz
warble) if a licence-clean source is preferred.
